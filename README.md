## ゲーム内容
下画面をタップ！タップ！とにかくタップ！超絶体力4鍵音ゲー！  
タップだけしたいゴリラ必見！あなたの3DSでもっとタップしよう！  

## 譜面導入
.tkj(譜面ファイル)とaudio.oggをtkjfilesフォルダに入れると動きます。

## 基本操作  
水色のノーツが流れてきたらレーンに合わせてタイミングよくタップするだけ！  

## 譜面命令
BPM変更  
```
#BPMCHANGE:(double)
```
拍子変更  
```
#MEASURE:(double)
```
スクロール変化
```
#SCROLL:(double)
```
